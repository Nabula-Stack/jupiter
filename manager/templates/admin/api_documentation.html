Admin Features & Quick Actions

When viewing a host in the admin panel, you have quick access to:

  • Host Information Dashboard
  • Virtual Machine Management
  • Network Configuration
  • Storage Management
  • Service Control Panel
  • Maintenance Mode Toggle
  • Quick Reboot Action

API Access Information

Base URL: /api/v1/

Full API Documentation: Available at /api/v1/docs/

API Version: 1.5.0

Description: Advanced API for managing ESXi Hosts, Virtual Machines, Networks, and Storage configurations via SSH

Management Features:

  • Host Management: Summary, licensing, reboot, shutdown, maintenance mode, service control
  • VM Management: Listing, creation, power operations, unregistration, cross-host migration
  • Network Management: vSwitch and port group configuration
  • Storage Management: Datastore listing, file explorer, disk management
